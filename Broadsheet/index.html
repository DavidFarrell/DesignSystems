THE ALASDAIR GRAY DESIGN SYSTEM · DAVID GÉROUVILLE-FARRELL
❦ A QUESTION OF MOMENT ❦

A self-contained design system modelled on the Glaswegian broadsheets and book-jackets of Alasdair Gray. Engraved double frames, sepia plates, fell-letter display, and a single oxblood accent for mottos. Posts are always single-column chapters; the three-column layout is reserved for the index. There are no icons, no shadows, no rounded corners, no gradients - the vocabulary is six Unicode glyphs and one frame.

❧ How to Use ❧

Load the three CSS files in this order, then write semantic markup with .v6-* classes:

❦ THE LOADING ORDER ❦
<link rel="stylesheet" href="fonts.css"/>
<link rel="stylesheet" href="src/tokens.css"/>
<link rel="stylesheet" href="src/components.css"/>
<body class="v6">
  <div class="v6-frame">
    <div class="v6-frame__inner">
      <span class="v6-frame__corner-bl">❦</span>
      <span class="v6-frame__corner-br">❦</span>
      <!-- masthead, body, end -->
    </div>
  </div>
</body>

The full rules are in README.md; the agent-skill manifest is in SKILL.md; the original handoff bundle's notes are in HANDOFF.md.
