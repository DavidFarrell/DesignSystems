01  ·  Principlesfoundations

Why this system exists

A homage to the Glaswegian broadsheets and book-jackets of Alasdair Gray. The system's job is to make a personal blog feel like a chapter from a strange, generous, hand-set book — without becoming a costume. Ornaments are real components, not decoration. The frame, the plate, the motto, and the drop cap are the system.

Frame the page.
Every screen sits inside a double engraved border with corner ornaments.
Read in one column.
Posts are single-column "chapters". 3-column layout is reserved for the index.
Plate, motto, drop cap.
These three carry the voice. Use them; don't substitute icons or color blocks.
02  ·  Color5 tokens · 1 filter

Palette

A cream paper with a 3px radial-dot tooth. A warm near-black ink. Oxblood for the accent. A sepia tint applied to all images via filter. That is the system; everything else is a softness of one of these.

Accent rules

Oxblood appears in: motto bars, the drop cap, the chapter label, ornament emphases, and link hover. Never as a fill on broad surfaces. A page with too much red stops being a broadsheet and becomes a warning sign.

04  ·  Spacing & layout4px base · 680px column

Rhythm & column

All spacing snaps to 4px. The reading column is 680px — narrower than V2, because the broadsheet voice asks for a more intimate measure. The frame margins (32px outer, 64×40px inner padding) are not negotiable.

Two layout modes

POST MODE — single column.
Reading column 680px max. Frame inset 32px / 64×40 padding. Drop cap on first paragraph. Inline plates centred at 600px max.
HOMEPAGE MODE — broadsheet 3-col.
Same outer frame, but the body splits into 3 columns with a 1px ink rule between them. Allowed only on the index. Never on posts.

07  ·  Patternspage templates

Two pages, one frame

The system supports the index (broadsheet 3-col) and the chapter/post (single column). The about page is treated as a frontispiece — same frame, same masthead, but the body becomes a single italic dedication.

INDEX (broadsheet)
Frame · masthead · 3-col body listing posts as cartouches with framed plate thumbnails. Column rule = 1px ink.
POST (chapter)
Frame · masthead · chapter label · title · dek · byline · hero plate · motto · single-column body · end-mark.
ABOUT (frontispiece)
Frame · masthead · large display title · italic dedication paragraph · author engraving plate · colophon.
08  ·  Do / Don'tedges of the system

The edges

DO
Use the broadsheet 3-column layout ONLY on the index. Posts must be single-column.
DON'T
Don't set post bodies in 2 or 3 columns. Long-form prose with inline images cannot be read across columns.
DO
Apply the sepia tint filter to every image. Frame every plate with the inset double border.
DON'T
Don't use raw screenshots without the plate frame. The frame is what signals "this is part of the broadsheet."
DO
Reuse the six ornament glyphs (❦ ❧ ☙ ❦❦❦ — — ).
DON'T
Don't add new ornaments, dingbats, or icon fonts. The Unicode set is the system's set.
09  ·  Using the systemhandoff

Building a new chapter

  1. Load tokens.css then components.css.
  2. Wrap the page in <body class="v6">.
  3. Wrap the chapter in .v6-frame > .v6-frame__inner with two .v6-frame__corner-bl/br spans for bottom corners.
  4. Inside: .v6-masthead → .v6-title → .v6-dek → .v6-byline → hero .v6-plate.v6-plate--hero → .v6-motto → .v6-body → .v6-end.
  5. Inside .v6-body use .v6-h2, .v6-h3, .v6-quote, .v6-plate, .v6-code-frame as needed.
  6. For the index page only, replace the body with a .v6-broadsheet wrapper.

If you need a new component, ask: does it fit the three principles (frame the page, read in one column, plate-motto-dropcap)? If yes, propose it. If no, you probably want a different system.
