# Brand Assets

This system has **no graphic logo files, no icons, no illustrations, and no photography**. The brand is expressed entirely through type, frames, and ornament glyphs.

## What lives here

- `wordmark.html` — the closest thing to a logo: the blog title set in IM Fell English with the small-caps tagline below. Use this as a self-contained masthead for any external embed (e.g. a favicon-rendering page, a deck title slide).

## Glyph reference (paste-as-text)

| Glyph | Code | Use |
|---|---|---|
| `❦` | U+2766 | Frame corners, motto bookends |
| `❧` | U+2767 | Section heading bookends, caption close |
| `☙` | U+2619 | Caption open |
| `—` | U+2014 | Em-rule, brackets H3 |
| `·` | U+00B7 | Middle dot, list separator in small-caps rows |

## What does NOT belong here

- No SVG icon set. No Lucide / Heroicons / Material / FontAwesome.
- No emoji.
- No PNG icons.
- No social-media glyphs. Set the names in small caps instead: `MASTODON · BLUESKY · RSS`.
- No raster photography of any kind. If imagery is needed, it goes through a `.v6-plate` with the sepia tint filter applied.

If a future component genuinely needs a pictorial mark, it should be drawn as line art in the engraving tradition, sepia-tinted, framed in a plate. Until that exists, the wordmark is the brand mark.
