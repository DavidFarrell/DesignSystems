Brutalist Editorial · 4 takes
System 01 / Brutalist
A1
Dark + electric yellow

Tabloid

Newspaper-masthead energy. Maximum contrast, electric accent, ticker strips, dense rules-and-rows. The Yellowball-adjacent take, pushed harder into print/editorial.
System 02 / Brutalist
A2
Cream + hot magenta

The Quarterly

Cream paper stock + hot magenta accent on warm ink. Asymmetric magazine grid. The "highlighted phrase" technique — magenta or black slabs behind key words.
System 03 / Brutalist
A3
Purple + fluoro yellow + pink

Riso Zine

Two-spot-colour overprint feel: deep purple bg + acid yellow + fluorescent pink halftone. Big oversized numbers, registration-mark accents, halftone dot textures.
System 04 / Brutalist
A4
White + tabloid red diagonal

Sports Tabloid

White paper, jet-black ink, single hot orange-red. Massive condensed type. Diagonal slash band that cuts across the page. Track-and-field energy.
Manifesto Poster · 3 takes
System 05 / Manifesto · ORIGINAL
C1
Terracotta + lemon + sky

Colour Blocks

Italic Fraunces display on warm bone. Coloured tiles for stats, manifesto block on ink, editorial spread for case studies. Original terracotta palette restored for take 1.
System 06 / Manifesto
C3
Magenta + cobalt + acid lime

Saturated Blocks

Same colour-block layout as C1 but with PUNCHIER colours — hot magenta + cobalt + acid lime + ink black on warm cream. Big italic Fraunces. Maximum flair.
System 07 / Manifesto
C4
Tangerine + violet on bone

Postered & Playful

Off-white + electric tangerine + violet + black. Asymmetric ransom-note collage feel. Mixes italic Fraunces with bold sans display. Diagonal headlines, sticker badges.
