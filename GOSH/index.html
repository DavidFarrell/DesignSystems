TYPOGRAPHY

One family - GOSH Brave

All headings, body and UI chrome use the GOSH Brave family. The fallback stack is Arial, Helvetica, sans-serif - matching the live site declaration. Visual hierarchy comes from weight (500 / 600 / 700 / 800) and size, never from family contrast.

RULES OF THE SYSTEM

The seven non-negotiables

  1. One family. GOSH Brave for everything.
  2. Body is 18px / line-height 1.4 / weight 600. Don't shrink it.
  3. Two button shapes only - rectangle and pill.
  4. One accent colour per surface. Cyan + lime + pink in the same panel is a clown car.
  5. Sentence case for headlines. Title Case for proper nouns only.
  6. No box-shadows. Layer with paper tiers, ruled lines, or coloured panels.
  7. Specificity over sentiment. "£25 funds an hour of research" beats "Your gift makes a difference."
