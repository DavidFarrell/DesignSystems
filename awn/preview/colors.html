A note on OKLCH

Awin's source CSS is authored in OKLCH - a perceptually-uniform colour space supported in modern browsers. We've kept the original OKLCH values as canonical because they round-trip without loss; the sRGB hex values shown above are visual approximations for designers using legacy tools. When in doubt, use the token (var(--awn-orange-default)) and let the browser resolve it.
